# 保護資料    
{: #insights_data_security}  

{{site.data.keyword.DRA_full}} 會在高度可用且安全的環境管理您的資料庫：
   * 資料採取靜態加密（GPFS、LUKS 及 Softlayer）和進行中加密（HTTPS 及 SSH）。用戶端及系統認證儲存在已加密的磁碟上。
   * 資料存取只限於需要資料以便支援及維護服務的使用者。
