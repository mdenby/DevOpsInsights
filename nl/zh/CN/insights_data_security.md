# 保护数据安全    
{: #insights_data_security}  

{{site.data.keyword.DRA_full}} 在高可用性的安全环境中托管您的数据库：
   * 数据在静态和动态状态下都是加密的（静态加密：GPFS、LUKS 和 Softlayer；动态加密：HTTPS 和 SSH）。客户机和系统凭证存储在加密的磁盘中。
   * 仅限那些需要使用数据来支持和维护服务的用户才能访问数据。
